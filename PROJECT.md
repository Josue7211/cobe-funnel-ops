# Project

## Name

Creator Funnel Ops

## Goal

Build a single lightweight web app that demonstrates the exact workflow described in the job post:

- ManyChat-style DM funnel
- Stripe handoff
- GHL-style call routing and no-show recovery
- revenue and tracking visibility

## Success criteria

- DM module clearly shows lead intake, tags, funnel stage, and automation outcomes
- recovery module clearly shows booked, reminded, no-show, and recovered states
- dashboard clearly shows revenue, KPIs, and event naming discipline
- repo includes plain-English docs and a week-one proposal
- build runs locally with no paid integrations required

## Non-goals

- Real Instagram authentication
- Full ManyChat clone
- Full GHL clone
- Production Meta CAPI posting
- Multi-user auth

## Audience

- Hiring manager
- interviewer
- operator who wants to understand how the workflow saves time and influences revenue
