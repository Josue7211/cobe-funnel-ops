# Week One For COBE

## What I would ship first

### 1. DM sprint funnel audit

- map current DM entry points
- normalize tags and offer names
- identify the shortest path from DM to Stripe checkout

### 2. No-show recovery pass

- define call owner routing rules
- standardize no-show tags and recovery timing
- add one-click rebook and proof-based follow-up

### 3. Revenue visibility

- stand up a daily dashboard pulling Stripe, funnel stage counts, and recovery performance
- define event naming for `Lead`, `Schedule`, and `Purchase`

## Why this order

This sequence closes the biggest gaps fastest:

- speed to lead
- revenue leakage from missed calls
- visibility into what is actually working
