COBE operator console

DM intake, recovery, and revenue visibility in one operator surface.

COBE Funnel Ops is a same-origin React + Express console for creator funnel teams. It keeps DM qualification, no-show recovery, payload validation, and operator audit trails in one live system.

Open live app View repo
Production https://cobe.aparcedo.org
Runtime contract One origin. Express serves the SPA and `/api/*` together.
Local start `npm install` then `npm run dev`

What it includes

DM Workbench

Lead intake, qualification, routing, and checkout handoff.

Recovery Branch

No-show escalation, rebooking, and recovery-state visibility.

Payload Lab

Webhook payloads, relay checks, and revenue event naming.

Systems Drawer

Inbox, rules, metrics, and operator triage surfaces.

Run it locally

  • Use Node 22+.
  • Install dependencies with `npm install`.
  • Start the same-origin dev stack with `npm run dev`.
  • Build production assets with `npm run build`.
  • Run the production-like server with `npm start`.

GitHub Pages is a companion page only. The real operator runtime lives at https://cobe.aparcedo.org.
