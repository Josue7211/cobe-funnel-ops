# Design System

## Direction

- Personality: Precision & Density
- Foundation: blue-black operator console
- Depth: borders-first with restrained glow
- Variation: retro-futurist terminal polish

## Tokens

### Spacing

- Base: 4px
- Scale: 4, 8, 12, 16, 24, 32

### Radius

- Outer panels: 10px
- Inner panels: 8px
- Controls: 8px

### Typography

- Display: narrow sans
- Body: sans
- Labels + data: mono

### Colors

- `--bg-0`: base canvas
- `--bg-1`: rail surface
- `--bg-2`: active surface
- `--line`: default border
- `--cyan`: queue / funnel work
- `--green`: delivered / healthy
- `--amber`: tests / revenue / live actions
- `--red`: recovery / risk
- `--magenta`: automation / Meta / intelligence

## Patterns

### Header Strip

- Single horizontal command frame
- Segmented metrics in one row
- No detached hero cards

### Queue Item

- 8px radius
- Left-aligned text
- Topline metadata + one action note
- Active state uses semantic edge + brighter surface

### Workspace Panel

- One title
- One status strip
- One action rail
- One primary content region

### Right Rail Module

- Compact list modules with clear status tags
- Prioritize event recency and operator actions over marketing-style summaries
