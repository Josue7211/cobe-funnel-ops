# Phase 6 Plan 02: Guided walkthrough

## Goal

Build a concise guided walkthrough that maps the job post skills to the app surfaces.

## Tasks

- Add a walkthrough panel that names the core interview skills and the corresponding live features.
- Make the walkthrough help the user jump between queue, workflow, tools, and reporting views.
- Keep the narrative short enough to use live in an interview.

## Verification

- The product can explain itself without external notes.
- The walkthrough references the exact job-post requirements.
- The guided flow feels like part of the console, not a separate presentation.
