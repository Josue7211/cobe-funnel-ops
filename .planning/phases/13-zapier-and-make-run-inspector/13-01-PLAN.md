# 13-01: Add Run Inspector Surface

## Goal
Show Zapier and Make runs in an inspectable view.

## Depends on
- Phase 12

## Requirements
- ORCH-01

## Success Criteria
1. Runs show trigger, branch, actions, status, retries, and downstream effects.
2. The inspector is readable in the operator console.

## Plan
- Render a run inspector in the automation rail.
- Include trigger, branch, replay, retry, and downstream-effect summaries.
- Tie the view to the current lead and selected connector context.
- Keep the inspector narrow enough to read without losing the queue, replay, or evidence rails.
