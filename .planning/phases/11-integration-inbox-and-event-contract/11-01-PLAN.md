# 11-01: Define Shared Integration Event Shape

## Goal
Define one shared inbound event shape for connector activity.

## Depends on
- Existing queue, timeline, audit, and connector state surfaces

## Requirements
- EVT-01

## Success Criteria
1. The event shape covers source, target, type, payload summary, status, and timestamps.
2. The shape is reusable across inbox, timeline, queue, and audit flows.

## Plan
- Review current event sources.
- Define the canonical event contract.
- Map the shape to the existing domain objects.
