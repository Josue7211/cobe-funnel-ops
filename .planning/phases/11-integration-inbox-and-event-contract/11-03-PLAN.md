# 11-03: Map Events Into Product State

## Goal
Use inbound events to mutate queue, timeline, and audit state.

## Depends on
- 11-01 and 11-02

## Requirements
- EVT-03

## Success Criteria
1. Lead timelines and queue state come from the same event flow.
2. Audit logs reflect the same mutations.

## Plan
- Wire the event flow into state updates.
- Reconcile queue and timeline views.
- Keep mutations explicit and traceable.
